## Source Quality Notes

The following sources from the NotebookLM deep dive are **not recommended** for inclusion in the proposal — they are low-quality, paywalled without accessible versions, or not peer-reviewed:

- Lie detector blog (ref 22 in deep dive) — exclude
- Science.gov topic aggregator (ref 23) — exclude
- NBCC journal volume listing without specific article (ref 24) — exclude
- bioRxiv preprint on Alzheimer's mouse model (ref 18) — use the PMC-published version (ref 19) if citing hippocampal-prefrontal dynamics

The HKS Architects (2021) report is grey literature from an architecture firm, not peer-reviewed. It is useful for the neurogenesis/enriched environment argument but should be supplemented with a peer-reviewed source (Bhattacharya et al., 2025) for the same claim.
